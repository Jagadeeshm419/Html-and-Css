• FIRST

    Computer stands for Common Oriented Machine Particularly Used for Trade, Education and Research. Charles Babbage is called as father of the computer. There are three parts of computer, such as ALU, Memory and Control unit. ALU stands for Arithmetic and Logical Unit. Memory is a storage device. Control unit is used to control the physical components of a system.
